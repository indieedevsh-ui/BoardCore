# BoardCore

Projekt gry planszowej BoardCore (iOS) oraz lekka strona (HTML/CSS/JS).

## Uruchomienie (bez serwera)

Otwórz plik `index.html` dwuklikiem lub przeciągnij go do przeglądarki. Nie potrzebujesz żadnego serwera lokalnego.

## Zakładki

- **Menu** — powitanie, wersja 1.0, kwadratowe kafelki z funkcjami, link do GitHub
- **O grze** — opis aplikacji

Skróty klawiaturowe: `1` (Menu), `2` (O grze).
